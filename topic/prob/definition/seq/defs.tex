\def\seqdef{%
for \ctx\ $\SSPT{\S}{(\XV,\X)}$,
  \ctext{$\SQ$ denote the set of all \seq{}s,}
where ``$\SQQ$'' denotes an arbitrary such \seq.

Let $1 \le \TR < \infty$ denote an arbitrary \trl.

Because \seq{}s are infinite (and cannot be written out), we use the notation
$$\SQI{\SQQ}{\TR}{\JASN{\ASN\XV\XX}} = 
\begin{cases}
  1 & \text{\trl\ $\TR$ of \seq\ $\SQQ$ has \asn\ $\JASN{\ASN\XV\XX}$}\\
  0 & \text{otherwise}
\end{cases}
$$
to describe a \seq\ $\SQQ \in \SQ$.
}
